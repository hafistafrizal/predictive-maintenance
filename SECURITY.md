# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | ✅ Yes              |
| 1.0.x   | ⚠️ Critical fixes only |

## Reporting a Vulnerability

Jika Anda menemukan celah keamanan pada proyek ini, mohon **jangan** melaporkannya secara publik melalui GitHub Issues.

Silakan kirimkan laporan melalui salah satu cara berikut:
- Email langsung kepada maintainer repo ini
- Gunakan fitur [GitHub Private Security Reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)

### Yang Perlu Dilaporkan
- Kerentanan pada koneksi database MySQL
- SQL Injection melalui input form
- Kerentanan pada file `.pkl` (deserialisasi tidak aman)

Kami akan merespons laporan dalam waktu **48-72 jam kerja**.
